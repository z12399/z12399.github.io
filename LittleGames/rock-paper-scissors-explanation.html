石頭剪刀布遊戲功能詳解

瞭解本遊戲的所有功能及實現原理

遊戲概述

石頭剪刀布是一款全球流行的手勢遊戲，通常用於在兩人之間解決小爭執或做出選擇。在我們的數字版本中，玩家與電腦對戰，通過選擇石頭、剪刀或布來進行比拼。

遊戲提供多種回合選擇，讓玩家可以選擇3、5、7或10回合的比賽。每個回合的勝負將被記錄，最終以總分決定遊戲勝負。

遊戲規則

石頭剪刀布的規則簡單明了：

  • 石頭 (✊) 勝過 剪刀 (✌️)：石頭可以砸鈍剪刀
  • 剪刀 (✌️) 勝過 布 (✋)：剪刀可以剪斷布
  • 布 (✋) 勝過 石頭 (✊)：布可以包裹石頭
  • 相同的選擇會導致平局

以下是規則的表格表示：

電腦：石頭 電腦：布 電腦：剪刀
玩家：石頭 平局 電腦勝 玩家勝
玩家：布 玩家勝 平局 電腦勝
玩家：剪刀 電腦勝 玩家勝 平局

功能特點

  • 多回合制：可以選擇3、5、7或10回合的比賽
  • 動畫效果：選擇後的手勢會有輕微的動畫效果
  • 視覺反饋：勝利、失敗和平局有不同的顏色標識
  • 得分記錄：實時顯示玩家和電腦的得分
  • 遊戲歷史：記錄每一回合的選擇和結果
  • 重置功能：可以隨時重新開始遊戲
  • 響應式設計：在各種屏幕尺寸上都能良好顯示

技術實現

這款石頭剪刀布遊戲完全使用純HTML、CSS和JavaScript實現，沒有使用任何外部庫。以下是一些關鍵技術亮點：

隨機選擇生成

電腦的選擇是通過JavaScript的隨機數生成函數來確定的：

function getComputerChoice() {
    const choices = ['rock', 'paper', 'scissors'];
    const randomIndex = Math.floor(Math.random() * 3);
    return choices[randomIndex];
}

勝負判斷邏輯

遊戲使用簡潔的邏輯來判斷每回合的勝負：

function getResult(playerChoice, computerChoice) {
    if (playerChoice === computerChoice) {
        return 'draw';
    }
    
    if (
        (playerChoice === 'rock' && computerChoice === 'scissors') ||
        (playerChoice === 'paper' && computerChoice === 'rock') ||
        (playerChoice === 'scissors' && computerChoice === 'paper')
    ) {
        return 'win';
    }
    
    return 'lose';
}

CSS動畫效果

遊戲中的手勢有簡單的上下晃動動畫，通過CSS @keyframes實現：

.animate {
    animation: shake 0.5s;
}
@keyframes shake {
    0%, 100% { transform: translateY(0); }
    25% { transform: translateY(-10px); }
    50% { transform: translateY(0); }
    75% { transform: translateY(10px); }
}

遊戲策略

雖然石頭剪刀布常被視為純粹的運氣遊戲，但實際上有一些策略可以提高獲勝機率：

  • 觀察模式：在現實生活中的遊戲中，觀察對手之前的選擇可能會發現一些模式
  • 心理戰術：人們往往在連續出同一選擇後會改變，這一點可以利用
  • 統計優勢：研究表明，許多人第一次會選擇石頭，所以首輪選擇布可能有優勢
  • 隨機性：對抗電腦時，最好的策略反而是保持隨機選擇，避免形成可預測的模式

請注意，在我們的遊戲中，電腦是完全隨機選擇的，所以沒有確定的"必勝"策略。
