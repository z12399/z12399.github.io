1. 游戏概述

围墙小游戏是一个简单而有挑战性的策略游戏。玩家需要通过点击格子建立墙来阻止中心物体逃向边缘。每建立一座墙，物体就会移动一步，它总是试图找到最短的路径逃向边缘。如果物体到达边缘，游戏失败；如果玩家成功将物体完全围住，则游戏胜利。

游戏的主要功能包括：

  • 创建一个10x10的网格游戏区域
  • 在中心位置放置一个物体
  • 允许玩家点击空格子建立墙
  • 物体每次会向最近的边缘移动
  • 判断游戏胜负条件
  • 记录墙的数量和游戏状态

3. JavaScript实现

3.1 游戏配置和状态

// 游戏配置
const config = {
    gridSize: 10, // 10x10 网格
    cellSize: 40, // 单元格大小
    wallColor: '#8B4513', // 墙的颜色
    objectColor: '#FF0000', // 物体颜色
    emptyColor: '#FFFFFF', // 空格子颜色
    gridLineColor: '#CCCCCC', // 网格线颜色
    moveDelay: 300 // 物体移动延迟（毫秒）
};

// 游戏状态
let gameState = {
    grid: [], // 存储网格状态：0=空，1=墙
    objectPos: {}, // 物体位置
    wallCount: 0, // 已建墙数
    gameOver: false, // 游戏是否结束
    win: false // 是否胜利
};

游戏初始化时，我们定义了两个主要对象：

  • config: 包含游戏的配置参数，如网格大小、颜色等
  • gameState: 维护游戏的当前状态，包括网格、物体位置和游戏状态等

3.2 初始化游戏

function initGame() {
    // 重置游戏状态
    gameState.grid = [];
    gameState.wallCount = 0;
    gameState.gameOver = false;
    gameState.win = false;
    
    // 初始化网格
    for (let i = 0; i < config.gridSize; i++) {
        gameState.grid[i] = [];
        for (let j = 0; j < config.gridSize; j++) {
            gameState.grid[i][j] = 0; // 所有格子初始为空
        }
    }
    
    // 初始化物体位置在中心
    const center = Math.floor(config.gridSize / 2);
    gameState.objectPos = { x: center, y: center };
    
    // 更新UI
    updateUI();
    
    // 绘制初始状态
    drawGame();
}

初始化游戏的主要步骤：

  1. 重置游戏状态变量
  2. 创建一个二维数组表示网格，初始为全空
  3. 将物体放在网格中心
  4. 更新用户界面和绘制游戏

3.3 绘制游戏

function drawGame() {
    // 清空画布
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    
    // 绘制网格
    for (let i = 0; i < config.gridSize; i++) {
        for (let j = 0; j < config.gridSize; j++) {
            // 绘制单元格
            if (gameState.grid[i][j] === 1) {
                // 墙
                ctx.fillStyle = config.wallColor;
            } else {
                // 空格子
                ctx.fillStyle = config.emptyColor;
            }
            
            ctx.fillRect(
                j * config.cellSize, 
                i * config.cellSize, 
                config.cellSize, 
                config.cellSize
            );
            
            // 绘制网格线
            ctx.strokeStyle = config.gridLineColor;
            ctx.strokeRect(
                j * config.cellSize, 
                i * config.cellSize, 
                config.cellSize, 
                config.cellSize
            );
        }
    }
    
    // 绘制物体
    if (!gameState.gameOver || gameState.win) {
        const x = gameState.objectPos.x;
        const y = gameState.objectPos.y;
        
        ctx.fillStyle = config.objectColor;
        ctx.beginPath();
        ctx.arc(
            x * config.cellSize + config.cellSize / 2,
            y * config.cellSize + config.cellSize / 2,
            config.cellSize / 3,
            0,
            Math.PI * 2
        );
        ctx.fill();
    }
}

绘制游戏的主要步骤：

  1. 清空Canvas画布
  2. 遍历网格，绘制空格子和墙
  3. 绘制网格线形成网格效果
  4. 在物体当前位置绘制一个圆形表示物体

3.4 物体移动逻辑

function moveObject() {
    if (gameState.gameOver) return;
    
    const x = gameState.objectPos.x;
    const y = gameState.objectPos.y;
    
    // 确定可能的移动方向（不包括有墙的格子）
    const possibleMoves = [];
    const directions = [
        { dx: 0, dy: -1, name: '上' },
        { dx: 1, dy: 0, name: '右' },
        { dx: 0, dy: 1, name: '下' },
        { dx: -1, dy: 0, name: '左' }
    ];
    
    for (const dir of directions) {
        const newX = x + dir.dx;
        const newY = y + dir.dy;
        
        // 检查新位置是否有效（在网格内且不是墙）
        if (
            newX >= 0 && newX < config.gridSize &&
            newY >= 0 && newY < config.gridSize &&
            gameState.grid[newY][newX] === 0
        ) {
            // 计算该方向到边缘的距离
            let distanceToEdge;
            if (dir.name === '上') {
                distanceToEdge = newY;
            } else if (dir.name === '右') {
                distanceToEdge = config.gridSize - 1 - newX;
            } else if (dir.name === '下') {
                distanceToEdge = config.gridSize - 1 - newY;
            } else { // 左
                distanceToEdge = newX;
            }
            
            // 添加到可能的移动方向，包含方向和到边缘的距离
            possibleMoves.push({
                dx: dir.dx,
                dy: dir.dy,
                distance: distanceToEdge
            });
        }
    }
    
    // 如果没有可能的移动方向，游戏胜利（物体被完全围住）
    if (possibleMoves.length === 0) {
        gameState.gameOver = true;
        gameState.win = true;
        updateUI();
        return;
    }
    
    // 根据到边缘的距离排序，选择最近的方向移动
    possibleMoves.sort((a, b) => a.distance - b.distance);
    const move = possibleMoves[0];
    
    // 更新物体位置
    gameState.objectPos.x += move.dx;
    gameState.objectPos.y += move.dy;
    
    // 检查游戏是否结束
    checkGameEnd();
    
    // 重绘游戏
    drawGame();
}

物体移动的核心逻辑：

  1. 确定物体当前可以移动的方向（不能穿墙）
  2. 对于每个可能的方向，计算该方向到达边缘的距离
  3. 根据距离排序，选择到边缘最短的路径移动
  4. 如果没有可移动的方向，说明物体被完全围住，游戏胜利
  5. 移动后检查游戏是否结束并重绘游戏

3.5 处理玩家点击建墙

function handleCanvasClick(event) {
    if (gameState.gameOver) return;
    
    // 获取点击位置相对于canvas的坐标
    const rect = canvas.getBoundingClientRect();
    const x = event.clientX - rect.left;
    const y = event.clientY - rect.top;
    
    // 转换为网格坐标
    const gridX = Math.floor(x / config.cellSize);
    const gridY = Math.floor(y / config.cellSize);
    
    // 检查点击位置是否有效（不能点击物体所在位置或已有墙的位置）
    if (
        gridX === gameState.objectPos.x && 
        gridY === gameState.objectPos.y
    ) {
        return; // 不能在物体位置建墙
    }
    
    if (gameState.grid[gridY][gridX] === 1) {
        return; // 不能在已有墙的位置重复建墙
    }
    
    // 建墙
    gameState.grid[gridY][gridX] = 1;
    gameState.wallCount++;
    
    // 更新UI
    updateUI();
    
    // 重绘游戏
    drawGame();
    
    // 延迟移动物体
    setTimeout(moveObject, config.moveDelay);
}

处理点击建墙的流程：

  1. 获取用户点击在Canvas上的坐标
  2. 转换为网格坐标
  3. 验证位置是否有效（不能在物体位置或已有墙的位置建墙）
  4. 在点击位置创建墙并增加墙的计数
  5. 更新界面并重绘游戏
  6. 短暂延迟后让物体移动，增加游戏的流畅感

4. 游戏状态判断

function checkGameEnd() {
    const x = gameState.objectPos.x;
    const y = gameState.objectPos.y;
    
    // 检查是否到达边缘
    if (x === 0 || x === config.gridSize - 1 || y === 0 || y === config.gridSize - 1) {
        gameState.gameOver = true;
        gameState.win = false;
        updateUI();
        return true;
    }
    
    // 检查是否被完全围住
    const directions = [
        { dx: 0, dy: -1 }, // 上
        { dx: 1, dy: 0 },  // 右
        { dx: 0, dy: 1 },  // 下
        { dx: -1, dy: 0 }  // 左
    ];
    
    let surrounded = true;
    for (const dir of directions) {
        const newX = x + dir.dx;
        const newY = y + dir.dy;
        
        // 如果任意一个方向不是墙，说明没被完全围住
        if (gameState.grid[newY][newX] === 0) {
            surrounded = false;
            break;
        }
    }
    
    if (surrounded) {
        gameState.gameOver = true;
        gameState.win = true;
        updateUI();
        return true;
    }
    
    return false;
}

游戏结束条件的判断：

  1. 如果物体到达网格边缘，游戏失败
  2. 如果物体四周都是墙，物体被完全围住，游戏胜利
  3. 根据结果更新游戏状态

5. 游戏扩展思路

可以考虑以下方式扩展游戏功能：

  • 设置不同难度等级，调整网格大小或物体移动速度
  • 添加更复杂的物体移动逻辑，例如随机移动或智能寻路
  • 增加多个物体同时存在，增加游戏难度
  • 添加特殊墙块，如临时墙或透明墙
  • 添加计分系统，根据使用的墙数量和围住物体的速度计算得分
  • 添加计时功能，限制游戏时间
  • 使用localStorage保存最佳记录
