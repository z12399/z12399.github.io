1. 游戏概述

这个猜数字游戏是一个简单的JavaScript互动游戏，它会随机生成一个1到100之间的数字，并让用户猜这个数字是多少。每次猜测后，游戏会提示数字是太大还是太小，并记录用户的猜测历史。

游戏的主要功能包括：

  • 随机生成1-100之间的数字
  • 验证用户输入
  • 判断猜测结果并给出反馈
  • 以醒目颜色显示提示信息
  • 记录并显示猜测历史
  • 重新开始游戏

2. HTML结构

游戏的HTML结构主要包括以下几个部分：

2.1 输入区域

<div>
    <input type="number" id="guessInput" min="1" max="100">
    <button onclick="checkGuess()">猜一猜</button>
</div>

<p id="message"></p>
<p id="attempts">已猜次数: 0</p>

这部分包含一个数字输入框、提交按钮和显示消息的区域。

2.2 历史记录表格

<div id="historyContainer">
    <h3>猜测历史记录</h3>
    <table id="historyTable">
        <thead>
            <tr>
                <th>次数</th>
                <th>猜测数字</th>
                <th>提示</th>
            </tr>
        </thead>
        <tbody id="historyBody">
            <!-- 这里将动态填充猜测历史 -->
        </tbody>
    </table>
</div>

这个表格用于显示用户的猜测历史，包括每次猜测的次数、猜测的数字和提示信息。

3. CSS样式

为了使提示信息更加醒目，我们使用CSS定义了不同颜色的提示样式：

.hint-smaller {
    color: blue;
    font-weight: bold;
}
.hint-larger {
    color: red;
    font-weight: bold;
}
.hint-correct {
    color: green;
    font-weight: bold;
}

这样，当猜测的数字：

  • 太小时，提示会以蓝色粗体显示
  • 太大时，提示会以红色粗体显示
  • 猜对时，提示会以绿色粗体显示

4. JavaScript实现

4.1 初始化游戏

// 游戏逻辑
let randomNumber = Math.floor(Math.random() * 100) + 1;
let attempts = 0;
let guessHistory = [];
const messageElement = document.getElementById('message');
const attemptsElement = document.getElementById('attempts');
const historyBody = document.getElementById('historyBody');

游戏初始化时，我们：

  • 使用Math.random()生成一个1到100之间的随机数
  • 初始化猜测次数为0
  • 创建一个空数组用于存储猜测历史
  • 获取页面上的DOM元素引用

4.2 处理猜测逻辑

function checkGuess() {
    const guessInput = document.getElementById('guessInput');
    const userGuess = parseInt(guessInput.value);
    
    // 验证输入
    if (isNaN(userGuess) || userGuess < 1 || userGuess > 100) {
        messageElement.textContent = '请输入1到100之间的有效数字！';
        messageElement.style.color = 'red';
        return;
    }
    
    attempts++;
    attemptsElement.textContent = `已猜次数: ${attempts}`;
    
    // 记录本次猜测
    let hint = '';
    let hintClass = '';
    if (userGuess === randomNumber) {
        hint = '恭喜猜对了！';
        hintClass = 'hint-correct';
    } else if (userGuess < randomNumber) {
        hint = '太小了，再大一点！';
        hintClass = 'hint-smaller';
    } else {
        hint = '太大了，再小一点！';
        hintClass = 'hint-larger';
    }
    
    // 将本次猜测添加到历史记录
    guessHistory.push({
        attempt: attempts,
        guess: userGuess,
        hint: hint,
        hintClass: hintClass
    });
    
    // 更新历史记录表格
    updateHistoryTable();
    
    // 更新游戏状态
    if (userGuess === randomNumber) {
        messageElement.textContent = `恭喜你猜对了！答案就是 ${randomNumber}，你总共猜了 ${attempts} 次。`;
        messageElement.style.color = 'green';
        document.querySelector('button').textContent = '重新开始';
        document.querySelector('button').onclick = resetGame;
    } else if (userGuess < randomNumber) {
        messageElement.textContent = '太小了，再大一点！';
        messageElement.style.color = 'blue';
    } else {
        messageElement.textContent = '太大了，再小一点！';
        messageElement.style.color = 'blue';
    }
    
    guessInput.value = '';
    guessInput.focus();
}

这个函数处理用户的猜测，主要步骤包括：

  1. 获取并验证用户输入
  2. 增加猜测次数
  3. 根据猜测结果设置提示信息和对应的样式类
  4. 将本次猜测添加到历史记录
  5. 更新历史记录表格
  6. 更新游戏状态（包括是否猜对、是否需要继续猜测）
  7. 清空输入框并将焦点重新置于输入框

4.3 更新历史记录表格

function updateHistoryTable() {
    // 清空当前表格内容
    historyBody.innerHTML = '';
    
    // 添加每条猜测记录
    guessHistory.forEach(record => {
        const row = document.createElement('tr');
        
        // 添加次数单元格
        const attemptCell = document.createElement('td');
        attemptCell.textContent = record.attempt;
        row.appendChild(attemptCell);
        
        // 添加猜测数字单元格
        const guessCell = document.createElement('td');
        guessCell.textContent = record.guess;
        row.appendChild(guessCell);
        
        // 添加提示单元格
        const hintCell = document.createElement('td');
        hintCell.textContent = record.hint;
        hintCell.className = record.hintClass;
        row.appendChild(hintCell);
        
        // 将行添加到表格
        historyBody.appendChild(row);
    });
}

这个函数负责更新历史记录表格，主要步骤包括：

  1. 清空表格当前内容
  2. 遍历猜测历史记录数组
  3. 为每条记录创建表格行
  4. 添加次数、猜测数字和提示信息的单元格
  5. 为提示信息单元格添加对应的CSS类，使其显示为醒目颜色
  6. 将行添加到表格中

4.4 重置游戏

function resetGame() {
    randomNumber = Math.floor(Math.random() * 100) + 1;
    attempts = 0;
    guessHistory = [];
    attemptsElement.textContent = '已猜次数: 0';
    messageElement.textContent = '';
    document.querySelector('button').textContent = '猜一猜';
    document.querySelector('button').onclick = checkGuess;
    
    // 清空历史记录表格
    historyBody.innerHTML = '';
}

这个函数用于重置游戏，主要步骤包括：

  1. 生成新的随机数
  2. 重置猜测次数和历史记录
  3. 清空消息显示
  4. 将按钮文本恢复为"猜一猜"
  5. 将按钮点击事件重新设置为checkGuess函数
  6. 清空历史记录表格

5. 功能扩展思路

如果想进一步增强这个游戏，可以考虑以下几点：

  • 添加难度级别（例如，更大的数字范围或有限的猜测次数）
  • 添加计时功能，记录完成游戏所需的时间
  • 添加最佳成绩记录，使用localStorage保存
  • 添加提示功能，例如告诉用户正确数字的范围
  • 添加音效反馈
  • 添加动画效果，使游戏更加生动
